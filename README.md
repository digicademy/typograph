# TypoGraph

TYPO3 extension for providing access to TYPO3 database table data via a GraphQL endpoint. Under the bonnet, it uses https://github.com/webonyx/graphql-php.

If you want to use this extension, you need to be sufficiently familiar with the basic concepts of GraphQL. You shold be sufficiently familiar with the basic concepts of GraphQL to use this extension. For example, you need to know how to create GraphQL schemas. If you want to learn more about GraphQL or need a refresher, have a look at the at https://graphql.org/learn/.

You can find details on installation, configuration and use, as well as how to set up example data and an example configuration in the `Documentation` folder.

As long as this extension has not been published, you can use the local rendering container to create a human-readable HTML version of the docs: https://docs.typo3.org/permalink/h2document:rendering
